JAVASRIPT

JavaScript (JS) is a programming language used to make websites interactive and dynamic.
It works together with:
HTML – structure
CSS – design
JavaScript – behavior

Examples of what JavaScript does:
1. Button clicks
2. Form validation
3. Pop-ups and alerts
4. Changing text, colors, images
5. Games and animations

WHERE JAVASCRIPT RUNS

i. Browser (Chrome, Firefox, Edge)
ii. Server (using Node.js)
iii. Mobile apps
iv. Desktop apps

JavaScript Output Methods
alert("Hello"); // popup
console.log("Hello"); // console
document.write("Hello"); // page output

Variables
Variables store data.
var name = "John"; // old
let age = 20; // modern
const pi = 3.14; // constant

Data Types
JavaScript supports the following data types:
1. Number – represents numeric values
2. String – represents text
3. Boolean – true or false
4. Undefined – variable declared but not assigned
5. Null – empty value
6. Object – collection of data

Applications
1. Form validation
2. Dynamic content
3. Interactive web pages

Conclusion

JavaScript is an important web programming language used to improve user interaction and functionality of websites.
